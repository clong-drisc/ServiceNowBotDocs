# Changing your instance user role

Change your PDI user role to either App Engine Studio Creator or Admin for different views and levels of access.

## Before you begin

Role required: none

## About this task

Depending on your learning or testing needs, you may need to change the user role for your PDI.

## Procedure

1.  To change your user role:

    1.  Open the **Account** menu to access the **My Instance** section.

    2.  Click the **Change User Role** instance action.

    3.  In the **Your instance actions** dialog, select **App Engine Studio Creator** or **Admin**.

        ![Option to change user role to admin](../image/change-role-pdi.png)

    4.  Click the **Change User Role** button.

    5.  A message indicates when your change user role request is completed successfully.

2.  To see the change in your user role:

    1.  Close any open PDI browser tabs.

    2.  Relaunch your PDI by clicking the **Start building** button in the **Account** menu.
