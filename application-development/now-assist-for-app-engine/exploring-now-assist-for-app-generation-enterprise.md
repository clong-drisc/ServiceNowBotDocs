# Exploring Now Assist for App Engine

Now Assist for App Engine enables you to create and deploy AI agents within custom applications.

## Now Assist for App Engine overview

Use the tools and components in Now Assist for App Engine to build AI agents and skills using the agentic AI capabilities of the ServiceNow AI Platform. Then, integrate the AI agents and skills into your custom App Engine applications.

## Now Assist for App Engine users

|User|Description|
|----|-----------|
|AI developer|AI developers build custom AI Agents and skills and embed them into custom applications.|
|Requesters and fulfillers|Requesters and fulfillers use custom applications to submit requests and complete tasks.|

## Now Assist for App Engine benefits

|Benefit|Users|
|-------|-----|
|Create custom solutions by building a custom skill or workflow for a custom application.|AI developers|
|Create custom AI agents for custom applications.|AI developers|
|Use the custom AI agents and skills added to custom applications to improve productivity and efficiency.|Requesters and fulfillers|
