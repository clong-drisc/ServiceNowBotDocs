# Zero Copy Connector for ERP field descriptions

Some tables of field descriptions in ERP Semantic Mining \(ERP-CM\) are too large to maintain in task topics. Find information on the those large tables in this section.

-   **[Zero Copy Connector for ERP connection and credentials field descriptions](erp-canvas-system-connection-form-details.md)**  
The Connection and Credential Alias modal contains connection and credential details that specify how Zero Copy Connector for ERP connects to the ERP \(Enterprise Resource Planning\) system.
-   **[Zero Copy Connector for ERP new system field descriptions](erp-canvas-create-new-system-descriptions.md)**  
The new system form in Zero Copy Connector for ERP contains information on connection details for the ERP \(Enterprise Resource Planning\) system.
-   **[Zero Copy Connector for ERP software info tab field descriptions](erp-canvas-system-info-details.md)**  
The software info tab in Zero Copy Connector for ERP contains information about the selected ERP \(Enterprise Resource Planning\) system.
-   **[Zero Copy Connector for ERP system list field descriptions](erp-canvas-system-list-descriptions.md)**  
The Create new system form in Zero Copy Connector for ERP contains information on connection details for the ERP \(Enterprise Resource Planning\) system.
-   **[Zero Copy Connector for ERP ERP model table field descriptions](erp-canvas-erp-data-model-table-fields.md)**  
The **Entity fields** tab for an ERP \(Enterprise Resource Planning\) model in Zero Copy Connector for ERP displays the table fields that are included in the ERP model.
-   **[Zero Copy Connector for ERP clone model field descriptions](erp-canvas-clone-model-fields.md)**  
The Clone ERP \(Enterprise Resource Planning\) model form in Zero Copy Connector for ERP enables you to clone an ERP model so you can edit the new model.
-   **[Zero Copy Connector for ERP new model field descriptions](erp-canvas-new-model-descriptions.md)**  
The new model form in Zero Copy Connector for ERP contains details for the ERP \(Enterprise Resource Planning\) model.
-   **[Zero Copy Connector for ERP remote table form field descriptions](erp-canvas-remote-table-descriptions.md)**  
The Remote table form in Zero Copy Connector for ERP enables you to create and edit remote tables in the ERP \(Enterprise Resource Planning\) model.
-   **[Zero Copy Connector for ERP extraction table field descriptions](erp-canvas-extraction-table-descriptions.md)**  
The Extraction table form in Zero Copy Connector for ERP enables you to create and edit extraction tables in the ERP \(Enterprise Resource Planning\) model.
-   **[Zero Copy Connector for ERP scheduled extraction field descriptions](erp-data-hub-scheduled-extraction-field-descriptions.md)**  
The Scheduled extraction form in Zero Copy Connector for ERP enables you to create and edit jobs to extract data at regular intervals.
-   **[Zero Copy Connector for ERP Monitor field descriptions](erp-data-hub-monitor-field-descriptions.md)**  
The Zero Copy Connector for ERP Monitor enables you to track ERP transactions and their progress.
