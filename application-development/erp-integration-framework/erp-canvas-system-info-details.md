# Zero Copy Connector for ERP software info tab field descriptions

The software info tab in Zero Copy Connector for ERP contains information about the selected ERP \(Enterprise Resource Planning\) system.

For process details, see [View Zero Copy Connector for ERP software information](../task/view-erp-system-information.md).

This feature is available starting in the Yokohama Patch 3 \(May 2025\) release.

|Field|Description|
|-----|-----------|
|SAP system info|Summary of SAP version, supported version, and database system.|
|Is unicode|If this option is selected, the system supports the international standard character set.|
|Machine type|Identification number of the machine type, such as server or VM, on which the SAP system is installed.|
|Kernel release|Identification number of the kernel \(core SAP component\) release installed on this system.|
|Operating system release|Identification number of the operating system, such as Linux or Windows, release on which the SAP system is installed.|
|ERP system|Name of the ERP system.|
|Node name|Name of server node that this system is using.|
|Kernel patch level|Identification number of the kernel patch installed on this system.|
|Hot package level|Identification number of the hot package \(corrections or enhancements to standard SAP code released sporadically between versions\) installed on this system.|
|SAP version|Identification number of the specific release of SAP installed on this system.|
|SAP system ID|Identification number of the SAP system within an organization that this system is using.|
|Supported SAP versions|Identification number of all releases of SAP supported on this system.|
|Base release|Identification number of the base release of SAP installed on this system.|
|Operating system|Operating system, such as Linux or Windows, on which the SAP system is running.|
|Database system|Database system, such as SAP HANA Database \(HDB\), the SAP system is using.|
|Supported database|Databases, such as HANA 1.0 or HANA 2.0 that the SAP system supports.|
