# Use a spreadsheet to add data

Add data tables and records to your application by uploading a Microsoft Excel spreadsheet.

Use the following procedures to upload a spreadsheet and use it to create data tables and import data records. Use the [Import a spreadsheet](spreadsheet-importing-spreadsheet.md) procedure first to upload your spreadsheet, and then create data tables from your spreadsheet import as needed.

-   **[Import a spreadsheet](spreadsheet-importing-spreadsheet.md)**  
Upload a Microsoft Excel spreadsheet and use it to create your application data model.
-   **[Create new table from spreadsheet import](spreadsheet-create-new-table.md)**  
Create new tables directly from a Microsoft Excel spreadsheet import.
-   **[Create new table from extensible table and spreadsheet import](spreadsheet-create-new-ext-table.md)**  
Create a new table by extending an existing table, and then using a Microsoft Excel spreadsheet import to update or customize it.
-   **[Modify existing table using spreadsheet import](spreadsheet-import-existing-table.md)**  
Use a Microsoft Excel spreadsheet import to update an existing application table.
