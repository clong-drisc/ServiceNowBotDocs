# Remove an app from App Engine Studio

If you have the appropriate permissions, you can remove an application from App Engine Studio \(AES\).

## Before you begin

Role required: admin, delete application, or delegated developer with permissions. For more information, see [Delegate developers using AES](../concept/aes-app-dev-workflow.md).

## About this task

Delegated developers can delete applications based on their permissions. To request delete permissions, contact your administrator.

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **App Engine Studio**.

2.  From the My Apps page, open your application.

3.  In your application, select the Edit application properties icon \(![Edit application properties icon](../image/edit-application-properties-purple.png)\).

4.  Select **Delete application**.

5.  On the dialog box, enter `delete` and then select **Delete**.
