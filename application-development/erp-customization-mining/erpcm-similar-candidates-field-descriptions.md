# ERP-CM similar candidates field descriptions

On a candidate record in ERP Semantic Mining \(ERP-CM\) the **Similar candidates** tab displays information about similar candidates for replatforming ERP \(Enterprise Resource Planning\) apps.

For process details, see [View and work with candidate details in ERP-CM](../task/erpcm-view-work-with-candidate-details.md).

<table id="table_ozn_3z5_jyb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Candidate

</td><td>

Name of the application candidate, which is the record number on the ServiceNow AI Platform.

</td></tr><tr><td>

Similar candidate

</td><td>

Record number of the similar candidate, which is automatically generated by the ServiceNow AI Platform.

</td></tr><tr><td>

Short description

</td><td>

Brief description of the similar candidate. For example, the tables the similar candidate reads.

</td></tr><tr><td>

ERP application

</td><td>

Name of the ERP application on the ERP system.

</td></tr><tr><td>

ERP module

</td><td>

Functional business area of the ERP system, such as Sales.

</td></tr><tr><td>

Similarity

</td><td>

Numeric score representing how similar the candidates are, from 0 through 1.A higher score, for example 0.94, indicates very similar candidates.

For more information, see [Recommendations and similar candidates in ERP-CM](../concept/erpcm-recommendations-and-similar-candidates.md).

</td></tr><tr><td>

Similar candidates

</td><td>

Number of similar candidates.Select the value in this column to view a list of the additional similar candidates.

</td></tr><tr><td>

Updated

</td><td>

Date and time the similar candidate details were most recently updated.
