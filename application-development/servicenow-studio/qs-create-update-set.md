# Create an update set in ServiceNow Studio

Create an update set as you work in ServiceNow Studio.

## Before you begin

Role required: admin or delegated developer

## About this task

Create an update set without having to leave ServiceNow Studio. For more information about creating an update set on the ServiceNow AI Platform, see [Get started with update sets](../../system-update-sets/reference/get-started-update-sets.md).

You can also watch this short video on how to create an update set.

Video on how to create an update set in ServiceNow Studio 

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Open the application that you have changes for that need a new update set.

3.  At the bottom-left corner of the canvas, expand the update set dialog by selecting the current update set from the status bar.

    ![To open the update set dialog, select the current update set from the status bar.](../image/sn-studio-status-bar-update-set-dialog.png)

4.  Select **New**.

5.  Enter a name for the new update set.

6.  Select **Save**.

7.  Select **Apply**.

    The changes you made in your app go to the new update set.
