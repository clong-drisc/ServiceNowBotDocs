# Add a workspace to a custom application with Now Assist for app generation

Use Now Assist for app generation to add a workspace to a custom application.

## Before you begin

Role required: a now.assist.creator role and the now\_assist\_panel\_user role

**Note:** If you have users that only need to edit \(not create\) apps, they can be assigned the delegated\_developer, now\_assist\_panel\_user, and now.assist.creator roles.

To add a workspace to an application in Now Assist for app generation, the Now Assist Experience Generation skill must be active. For more information, see [Experience generation](../../../administer/ui-generation/task/generate-ui.md).

## About this task

As you're creating or updating custom applications, use Now Assist for app generation to add a workspace to your application. Specify what you want in the workspace or ask for recommendations.

**Note:** The following steps explain how to add a workspace to an existing application, but the process to add a workspace to an application that you're creating is similar.

## Procedure

1.  Navigate to **App Engine** &gt; **ServiceNow Studio**.

2.  Select the Now Assist icon.

    ![Now Assist highlighted in banner.](../images/app-generation-task-initiation-xsr2.png)

3.  In the Now Assist panel, select **Update an app**.

    ![Now Assist panel with update an app button highlighted.](../images/app-generation-task-update-app.png)

4.  Select the application to edit.

5.  Select **Submit**.

6.  Start the conversation and ask for a workspace.

    ![Now Assist panel with add a workspace request highlighted.](../images/app-generation-add-workspace1.png)

7.  Specify what you want or ask for suggestions.

    Now Assist for app generation can suggest a variety of elements to include in a workspace.

    ![Now Assist panel with recommendations for a workspace listed.](../images/app-generation-add-workspace2.png)

8.  Continue the conversation until the workspace contains what you need.

    ![Now Assist panel with elements specified for the new workspace.](../images/app-generation-add-workspace3.png)

9.  When you're ready, preview and save the application to view the workspace.

    For more information, see [Review and edit applications using Now Assist for app generation in ServiceNow Studio](sns-app-gen-review-apps.md).
