# Now Assist for app generation supported field types

Now Assist for app generation supports field types including choice, reference, and string.

App generation currently supports the creation of the following field types:

-   Choice
-   Currency
-   Date
-   Date/Time
-   Decimal
-   Integer
-   Reference
-   String
-   True/False

For a detailed list and description of all field types available in the platform, see [Field types reference](https://www.servicenow.com/docs/access?context=r_FieldTypes&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US).
