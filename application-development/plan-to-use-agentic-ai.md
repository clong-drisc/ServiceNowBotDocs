# Use agentic AI to build and edit applications

Use agentic AI to help create and edit applications. Agentic AI can also summarize what an app does and help you generate code.

## Create new applications with Now Assist for app generation

Expedite the initial development of an application by having a conversation with agentic AI. Describe what you need or ask Now Assist for app generation to help you with the business process you want in the application. Review the summary and preview the application for accuracy. Continue the conversation to refine the application. When ready, generate the application and verify the components in ServiceNow Studio. Modify the application in ServiceNow Studio as needed.

For more information, see [Generate apps with Now Assist for app generation within ServiceNow Studio](../../now-assist-app-gen/task/sns-app-gen-using-landing.md)

## Edit existing applications with Now Assist for app generation

Now Assist for app generation can also be used to modify existing applications. for example, extend application functionality by adding flows and automation, script includes, business rules, and other features.

For more information, see [Now Assist for app generation in ServiceNow Studio](../../now-assist-app-gen/concept/sns-now-assist-app-gen-landing.md).

## Summarize application purpose with Now Assist app summary generation

Use Now Assist app summary generation to have agentic AI quickly create a few sentences about what an application does. Copy the summary to use as the application description and use the summary to check for duplicate applications.

For more information, see [Now Assist for app summary generation in ServiceNow Studio](../../now-assist-app-summarize/concept/sns-now-assist-app-summarize-landing.md).

## Create a user experience with Now Assist for UI generation

Use UI generation to describe the table, chart types, and navigation you need and have agentic AI create an experience. Then, you review. If you accept, the experience is saved and you can make modifications. If you reject, you can continue the conversation to improve the experience.

For more information, see [Exploring UI generation](../../../administer/ui-generation/concept/exploring-ui-generation.md).

## Generate code with agentic AI

Use Now Assist for code generation to write custom scripts or iterate on scripts. Describe the goal of the code in the Code with Now Assist dialog box and view the code suggestion in the lines following your prompt. Suggestions are not added to your script until you accept them.

For more information, see [Exploring code generation](https://www.servicenow.com/docs/access?context=exploring-now-assist-code&version=yokohama&pubname=yokohama-api-reference&ft:locale=en-US).
