# Web analytics viewer \[web\_analytics\_viewer\]

View User Experience Analytics in the application navigator. Assigned users have viewer role permissions for User Experience Analytics for web applications.

-   core\_ui\_analytics\_viewer
-   now\_experience\_analytics\_viewer
