# Define texts for Notice and Explicit Opt-in user consent management policies

Edit the texts for Notice and Explicit Opt-in consent policies from the default text provided.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All**.

2.  In the navigator, enter `sys_ui_message.list` to open the Messages page.

3.  In the Key field enter either:

    -   `glide.analytics.consent.text.notice`
    -   `glide.analytics.consent.text.explicit_opt_in`
4.  Select the Key message to edit.

5.  In the message area, enter your updated text and select **Update**.
