# View data visualizations

A dashboard viewer has several ways to affect what a data visualization shows in runtime.

-   **[Select a group-by value in a data visualization as a viewer](select-group-runtime.md)**  
A viewer of a data visualization can select the value for grouping the data in the visualization.
-   **[Refresh a data visualization as a viewer](refresh-data-viz-runtime.md)**  
A viewer of a data visualization can refresh that data visualization without refreshing the page or having editing rights.
-   **[Editing columns on a Simple List](editing-columns-simple-list.md)**  
You can select which columns are shown when viewing a Simple List. You cannot select more columns than the maximum number that the list is configured to show.
