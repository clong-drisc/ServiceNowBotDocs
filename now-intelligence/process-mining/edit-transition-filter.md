# Edit a transition filter

Edit a transition filter if you want to make any changes to it.

## Before you begin

Role required: sn\_process\_optimization\_analyst, sn\_process\_optimization\_power\_user, or sn\_process\_optimization\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Process Mining Workspace**.

2.  Open a project for which you want to set the transition filter.

3.  Go to the **Analyst Workbench** tab.

4.  Apply filters including a transition filter in between.

    For information about how to apply filters, see [Create a transition filter manually](create-transition-manually.md).

5.  Open the **Applied Filters** list.

    ![Applied filters list](../image/edit-transition-filter.png)

6.  Select the edit icon \(![Edit icon](../../user-exp-analytics/image/edit-icon.png)\) beside the transition filter.

    **Note:** You will be able to edit the transition filter only using the fields in the View on which the filter is applied.

    The transition filter form is displayed. For information on the form, see [Create a transition filter manually](create-transition-manually.md).

7.  Edit the filter as required.
