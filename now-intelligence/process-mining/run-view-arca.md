# Run automated root cause analysis reports

Execute automated root cause analysis to optimize your processes.

## Before you begin

Role required:sn\_process\_optimization\_admin, sn\_process\_optimization\_power\_user, or sn\_process\_optimization\_analyst

## Procedure

1.  Run automated root cause analysis from any of the following locations:

<table id="table_wqc_yz4_krb"><tbody><tr><td>

Open a project.

 On the **Summary and Insights** page, go to the **Improvement Opportunities** section. From the available improvement opportunities, select **Action** &gt; **Key contributors \(RCA\)**.

</td><td>

![Root cause analysis option in the Summary and Insights page](../image/arca-location-1a.png)

</td></tr><tr><td>

Open a project.

 On the **Analyst Workbench** page, select a node or connector.

 From the pop-up window, select **Investigate** &gt; **Key contributors \(RCA\)**.

</td><td>

![Root cause analysis option in Analyst Workbench](../image/arca-location-2a.png)

</td></tr></tbody>
</table>    The root cause analysis task will begin to run. After the analysis is complete, you will see a notification. You can select the scheduled task icon \(![Scheduled tasks icon](../image/scheduled-task-icon.png)\) on the right of the screen to see an update on the task's progress.

2.  When the root cause analysis completes, select **View result** in the scheduled tasks panel of the Analyst Workbench.

    You can select **View result** from the **Summary and Insights** page too.

    The root cause analysis result is similar to the following:

    ![Automated root cause analysis result](../image/arca-results.png)
