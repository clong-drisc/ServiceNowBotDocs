# Evaluating Process Mining

Evaluate Process Mining with some supported business processes.

This section covers the following sections.

-   **[Process Mining evaluation project for Incident Management](evaluation-pm-inci-manag.md)**  
Process Mining evaluation project for Incident Management is a limited version of Process Mining. It’s shipped as part of the Process Mining plugin and available for customers that aren’t entitled for Process Mining for ITSM. It enables you to familiarize with improving your process with Process Mining capability.
-   **[Process Mining evaluation project for Customer Service Management](evaluation-pm-csm.md)**  
Process Mining evaluation project for Customer Service Management \(CSM\) is a limited version of Process Mining. It’s shipped as part of the Process Mining plugin and available for customers that aren’t entitled for Process Mining for CSM. It enables you to familiarize with improving your process with Process Mining capability.
-   **[Process Mining evaluation project for Human Resources](evaluation-pm-hr.md)**  
Process Mining evaluation project for Human Resources \(HR\) is a limited version of Process Mining. It’s shipped as part of the Process Mining plugin and available for customers that aren’t entitled for Process Mining for HR. It enables you to familiarize with improving your process with Process Mining capability.
-   **[Run Process Mining evaluation project](../task/run-evaluation-project.md)**  
Run Process Mining evaluation project to familiarize with improving your process with Process Mining capability.
