# Setting improvement opportunity for projects

Set improvement opportunities for projects to find areas of improvement by which teams could optimize a process. You can either select from a list of rules available to you or set your own rules. Based on the rules set by you, you can view your areas of improvement.

You can set improvement opportunity using the Finding Builder or the Classic view from the following locations:

-   Creating or editing a project from the Classic view.
-   Creating or editing a project from Project Builder.
-   Adding improvement opportunity for a project from the Summary and Insights page.
