# Dashboard reference

Reference topics provide a list of roles and translatable fields for Platform Analytics dashboards.

-   **[Translatable fields in dashboard elements](../../performance-analytics/concept/translatable-fields-dashboard.md)**  
For fields that show the **Translatable turned on** badge, you can add translations of the field value in the Messages \[sys\_ui\_message\] table.
-   **[Platform Analytics dashboard cache](pa-dashboard-cache.md)**  
Changes to the layouts of Platform Analytics dashboards are stored in the par\_dashboard\_cache table. Whenever a dashboard is called, a new entry is made in this table. Delete entries in this table to regenerate the cache for troubleshooting.
-   **[Platform Analytics dashboard roles](pa-dashboard-roles.md)**  
Platform Analytics dashboards have few role restrictions. Editing rights are granted by sharing, independent of role.
-   **[Platform Analytics dashboard tables](../reference/par-dashboard-tables.md)**  
The following tables relate to Platform Analytics dashboards and can be accessed through scripts.
