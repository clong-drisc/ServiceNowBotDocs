# Export platform list to Excel

A list displays a set of records from a table. You can export information from lists in the classic environment to a spreadsheet.

## Before you begin

Role required: none, though you must have rights to view the list.

For more information, see [Lists in the classic environment](https://www.servicenow.com/docs/access?context=c_UseLists&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US).

## Procedure

1.  Navigate to the list you want to export.

2.  Right-click any column heading and select **Export** &gt; **Excel**.
