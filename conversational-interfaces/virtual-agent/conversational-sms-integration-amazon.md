# Integrating Conversational SMS with AWS End User Messaging

Install the Conversational SMS with AWS End User Messaging \(sn\_sms\_aws\_adapter\) plugin from the ServiceNow store to enable users to connect to customer support agents via SMS using the Amazon End User Messaging as the SMS provider.

To use conversational SMS with AWS End User Messaging, complete the following setup tasks.

|Task|Description|
|----|-----------|
|[Install the Conversational SMS Integration with AWS End User Messaging application](../task/install-conversational-sms-integration-with-amazon.md)|Install the plugin to host conversations with virtual or live agents.|
|[Set up and configure the conversational SMS Integration with AWS End User Messaging application](set-up-conversational-sms-integration-with-amazon.md)|Set up the integration on both AWS and ServiceNow platform to engage in SMS bot conversations and address customer queries.|
|[Using conversational SMS integration with AWS End User Messaging](using-conversational-sms-integration-with-amazon.md)|Enable customers to interact with agents using the SMS integration with Amazon End User Messaging.|

-   **[Install the Conversational SMS Integration with AWS End User Messaging application](../task/install-conversational-sms-integration-with-amazon.md)**  
You can install the Conversational SMS Integration with AWS End User Messaging application \(sn\_sms\_aws\_adapter\) from the ServiceNow store and integrate it with AWS End User Messaging to enable customer support agent conversations via an SMS channel.
-   **[Set up and configure the conversational SMS Integration with AWS End User Messaging application](set-up-conversational-sms-integration-with-amazon.md)**  
Integrate Conversational SMS with AWS End User Messaging to enable customers to chat with live or virtual agents
-   **[Using conversational SMS integration with AWS End User Messaging](using-conversational-sms-integration-with-amazon.md)**  
Enable a customer to converse with a live or virtual agent using the SMS integration with AWS End User Messaging.
