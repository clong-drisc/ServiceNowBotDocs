# Analyzing Virtual Agent performance

Analyze the performance of Virtual Agent through preconfigured dashboards.

A new build of Conversational Analytics dashboard on Platform Analytics experience is available starting in Washington DC. The dashboard offers many of the same visualizations in the form of indicators and breakdowns, with no dependency on User Experience Analytics for data aggregations. The dashboard also meets the compliance requirements of data regulated market environments, for example, Government Community Cloud \(GCC\), and thus FedRAMP authorized. See [Conversational Analytics dashboard in Platform Analytics experience](VA-dashboard-landing-page-pae.md) for more information.

The existing [Legacy - Conversational Analytics Dashboard](VA-dashboard-landing-page.md) will be deprecated in Yokohama.
