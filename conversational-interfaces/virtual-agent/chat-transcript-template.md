The conversation transcript .txt file contains information used to troubleshoot individual conversations.

Start time of the conversation.

</td></tr><tr><td>

Channel

</td><td>

Channel from which the conversation was initiated. For example, the Slack application, Microsoft Teams, and so on.

User Name

</td><td>

Name of the user who initiated the conversation.

</td></tr><tr><td>

User Status
