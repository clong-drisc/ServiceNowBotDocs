# Virtual Agent Designer user input and bot response controls in Slack

The Virtual Agent Designer user input controls and bot responses for creating conversation topics are supported in Slack conversations, including the table bot response and the card control that can display images.

## Virtual Agent Designer user input controls and bot responses

Starting with Version 2.0.0, the card and video bot response controls are supported in Slack conversations. The video control displays a thumbnail image of the video that users select to open the video in their preferred web browser.

For more information about user input controls and bot responses, see [Virtual Agent Designer user input controls](../reference/va-user-inputs.md) and [Virtual Agent Designer bot responses](../reference/va-bot-responses.md), respectively.

## Upload multiple file types

Starting from Version 5.0.2, you can upload multiple image files \(up to 10 files\) from your desktop or by providing the valid link for an image picker topic in Virtual Agent. You can also share multiple files \(up to 10 files\) to a Virtual Agent topic via a link or by uploading it from your desktop.

To upload multiple image files or other attachments during your conversation with the bot, select **Image file types** or **All file types** respectively.

**Note:** You can also share multiple files with a Live Agent from Slack.
