# Integrating ServiceNow Virtual Agent with Slack

Enables ServiceNow® Virtual Agent to integrate with the Slack application.

The Virtual Agent integration with Slack creates a one-to-one association between the ServiceNow instance that you are installing from and the Slack workspace in which the integration runs. By default, Virtual Agent supports one-click installation that integrates one Slack workspace with one ServiceNow instance, which uses the Now Virtual Agent by default.

-   **[Install the Conversational Integration with Slack](../task/install-slack-from-store.md)**  
You can install the Conversational Integration with Slack \(sn\_va\_slack\) application if you have the admin role.
-   **[Integrate ServiceNow Virtual Agent with Slack](../task/add-slack-snow-bot.md)**  
Add the Virtual Agent bot to your ServiceNow® instance to integrate with Slack.
-   **[Override Slack integration](../task/override-slack-install.md)**  
Override Virtual Agent integration with Slack.
-   **[Manage the ServiceNow Virtual Agent integration with Slack](../task/configure-va-slack.md)**  
Manage your Conversational Integration with Slack application, its branding, and customize the bot-specific messages in your ServiceNow instance after integration.
-   **[Remove ServiceNow Virtual Agent integration with Slack](../task/uninstall-virtual-agent-slack.md)**  
Remove the ServiceNow Virtual Agent integration from your ServiceNow instance to disassociate the Slack app.
