# Configure Dynamic Translation service integration credentials

Configure the credential and connection for your translation service.

## Before you begin

**Note:** The following content is only relevant to NLU conversations and live agent LLM conversations that use Dynamic Translation for Agent Chat.

Complete the [prerequisites for using Dynamic Translation in Virtual Agent](../concept/prereq-using-dt-va.md).

Role required: admin

## Procedure

1.  Create a credential for the translator service alias.

    For detailed information, see the following topics:

    -   [Create a credential for the MicrosoftTranslation alias](https://www.servicenow.com/docs/access?context=create-credential-ms-translation&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US)
    -   [Configure the credential for the GoogleTranslation alias](https://www.servicenow.com/docs/access?context=configure-credential-google&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US)
2.  Create a connection for the translator service alias.

    For detailed information, see the following topics:

    -   [Create a connection for the MicrosoftTranslation alias](https://www.servicenow.com/docs/access?context=create-connection-ms-translation&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US)
    -   [Configure the connection attributes for the GoogleTranslation alias](https://www.servicenow.com/docs/access?context=configure-connection-google&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US)
