# Configure push notifications

Push notifications enable the sending of unsolicited messages \(notifications\) from a ServiceNow instance to a specific registered user and an application that leverages the ServiceNow Mobile SDK framework.

You must set up push notifications on both your ServiceNow instance and within the application that will receive the notifications.

**Note:** For the current release, only the Virtual Agent service supports the receipt of push notifications.
