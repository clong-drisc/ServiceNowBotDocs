# Export a snapshot of a CI

You can export a snapshot of a CI from its timeline.

## Before you begin

The CI must be in the Configuration Item \[cmdb\_ci\] table or a descendant of this table. Auditing must be enabled for the table containing the CI.

Role required: admin.

## About this task

You can export a snapshot of the CI to an XML, PDF \(Portal\), or PDF \(Landscape\) format.

## Procedure

1.  Open the timeline for the CI.

2.  Select the bubble representing the time for which you want to export a snapshot of the CI.

3.  Click the export icon \(![](../image/ExportCIHistoryChanges.png)\).

4.  Select the file format to use for the export.

    You can download the file to your system for viewing.
