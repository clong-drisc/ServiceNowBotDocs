# SoftPIN authentication

Soft PIN is an authentication method that verifies a user's identity through a personal numeric PIN set up in advance, specifically designed to support AI Voice Agent authentication.

## Use case

Soft PIN is ideal for low-risk scenarios, such as verifying returning users accessing self-service features. It isn’t recommended for high-risk or privileged actions.

## Key strengths

The SoftPIN method offers the following advantages:

-   Fast and straightforward user experience.
-   Functions without internet or device-specific requirements.

## Important considerations

While SoftPIN is a secure and convenient authentication method, there are several considerations to keep in mind:

-   Susceptible to reuse, observation, and social engineering.
-   Not suitable as the sole authentication method for sensitive or privileged operations.

Soft PIN can be used for your specific use case. For detailed configuration instructions, see [Configure Soft PIN](../task/configure-soft-pin.md).
