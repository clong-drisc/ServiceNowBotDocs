# Create group filter criteria

Group filter criteria allows or denies user access based on the user group to which the user belongs.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Adaptive Authentication** &gt; **Filter Criteria** &gt; **Group Filter Criteria**.

2.  Click **New**.

3.  On the form, fill in these fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name to identify the group.|
    |Description|Short description of the group.|
    |Application|Scope of the application.|

    ![Group filter criteria form](../images/group-filter-criteria-2.png "Example group filter criteria record")

4.  From the **Groups for criteria** tab, double-click **Insert a new row**.

5.  Click the search icon ![search icon](../images/search.png), look up, and select a user group.

6.  Click the save icon ![save icon](../images/green-checkmark.png).
