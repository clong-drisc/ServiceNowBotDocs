# Activate data filtration

Learn how to activate data filtration on your instance.

## Before you begin

**Important:**

Starting with the Yokohama release, Data filtration is being prepared for future deprecation. It will be hidden and no longer activated on new instances. Consider using Security data filters to restrict access to records based on role, or security-attribute related assertions. For details see [Data filtration\(Legacy\)](../concept/data-filtration-landing.md).

Role required: security\_admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Plugins**.

2.  Use the search field to find the Data Filtration \(com.glide.data\_filtration\) plugin.

3.  Click **Install**, and then in the Activate Plugin dialog box, click **Activate**.
