# Data filtration\(Legacy\)

Use data filtration to control access to tables and records based on subject attributes when performing read queries.

**Important:**

Starting with the Yokohama release, Data filtration is being prepared for future deprecation. It will be hidden and no longer activated on new instances. Consider using Security data filters to restrict access to records based on role, or security-attribute related assertions. For details see [Data filtration\(Legacy\)](data-filtration-landing.md).
