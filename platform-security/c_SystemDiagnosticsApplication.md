# System Diagnostics module

The System Diagnostics application provides logs that relate to the platform.

These logs are available:

-   **Upgrade History**

    Tracks every upgrade to an instance.

-   **Slow Queries**

    Provides insight into how queries affect platform performance. See [Use a slow query log](https://www.servicenow.com/docs/access?context=t_UseASlowQueryLog&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US).
