# Use condition builder in reference lists

Use condition builder in reference lists to filter results.

## Before you begin

Role required: workspace\_user

## Procedure

1.  Select **Search for Record** icon ![icon search.](../../../use/navigation/image/IconSearch.png) in the record field.

2.  Select ![condition builder filter icon.](../image/condition-builder-filter.png)

3.  Add the conditions you would like to filter your record search by.![condition builder filter fields.](../image/condition-builder-filter-display.png)

4.  To add additional conditions, select **+ New condition set**.

5.  Select **Run**.
