# Make form templates available for multiple groups

Specify one or more groups that have access to the form template you created.

## Before you begin

Role required: admin

## Procedure

1.  In the contextual side panel of a record, select the Templates icon \(![Templates icon.](../image/templates-icon-new.png)\).

2.  In the Templates pane, select the Create Templates icon \(![Create Templates icon](../image/icon-plus-sign.png)\).

3.  Select the Groups field, and enter the groups you want this form to be available to.

    ![Multiple groups.](../image/template-groups.png)
