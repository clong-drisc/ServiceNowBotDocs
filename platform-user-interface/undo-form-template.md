# Undo a form template

Undo a form template to the state prior to the application of the template.

## Before you begin

-   Only the last form template added is reversed.
-   Form field values are reverted back to the state prior to the application of the template.
-   Work you've performed isn't changed or deleted by undoing a template.

Role required: agent

## Procedure

1.  Select the Templates icon \(![Template icon.](../image/templates-icon-new.png)\) in the Contextual Sidepanel.

2.  On the Templates pane, select Undo.

    ![Undo templates.](../../../get-started/servicenow-overview/image/templates-undo.png)

    A modal appears displaying changes to the form as a result of removing the template.

    ![Undo template changes.](../image/template-undo-changes.png)

3.  Select **Undo**.
