# Change form header levels

Customize your Workspace in UI Builder by hiding the first section title or changing the level of section headings that appear on forms.

## Before you begin

Role required: ui\_builder\_admin

## Procedure

1.  Open an existing Workspace page or [create a new page in UI Builder](https://www.servicenow.com/docs/access?context=create-page&version=yokohama&pubname=yokohama-application-development&ft:locale=en-US).

2.  To hide the first section title on a Workspace form:

    1.  Select a form on the Workspace page.

    2.  Navigate to the Form Controller, and select **Hide first section title**.

3.  To change a heading level:

    1.  Select a form on the Workspace page.

    2.  Select the header that you want to edit on the form, and navigate to the Form Controller.

    3.  Select a heading level from the Level menu.
