# Configure Next Experience notification preferences

Set up how you receive in-product and system notifications in Next Experience.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **User Menu** &gt; **Preferences** &gt; **Notifications**.

2.  Select the **General** tab.

3.  Configure your preferences for system notifications outside of the platform.
