# Personalize your Workspace list columns by dragging them to the order you want

Drag your list columns to where how you want them.

## Before you begin

Column personalization must be enabled.

Role required: agent

## Procedure

1.  Drag a column using your mouse.

    1.  In a Workspace list, select and hold the grab icon \(![List column icon.](../image/list-column-icon.png)\) in the column header of the column you want to drag.

    2.  Drag the column and place it between the two columns where you want to place it.

        ![Drag list columns.](../image/drag-columns-screenshot.gif)

2.  Drag a column using your keyboard.

    1.  In a Workspace list, select the column header of the column you want to drag and press **Enter** on your keyboard.

    2.  Press **Enter** again on your keyboard to select the grab icon \(![List column icon.](../image/list-column-icon.png)\).

    3.  Use your arrow keys to move the column to the location you want.

    4.  Press **Enter** on your keyboard to place the column.
