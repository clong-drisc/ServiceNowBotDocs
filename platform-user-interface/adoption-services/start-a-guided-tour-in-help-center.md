# Start a guided tour in the help center

Locate and start a guided tour in the help center on your instance.

## Before you begin

Role required: admin

## Procedure

1.  To access this tour again, select the ![show help icon.](../../../help-center/adoption-services/image/show-help-icon.png) icon in navigation bar to open the help center.

2.  **Note:** You only need to select the **Help articles** tab if a **What's New** tab is also available. Otherwise, the help icon should open in **Help articles**.

    Select the **Help articles** tab, and scroll to the bottom of the help center.

3.  Select the ![image.select-a-tour-2] button.

4.  Select a tour from the guided tours list.

5.  ![begin tour button.](../../../help-center/adoption-services/image/begin-tour.png)Select **Begin Tour** to start the guided tour.
