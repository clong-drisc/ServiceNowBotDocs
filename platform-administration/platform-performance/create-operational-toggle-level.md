# Create an operational toggle level to define thresholds

Create operation toggle levels so that you can define the threshold limits that your application can function within.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Run Level** &gt; **Operational Toggle Levels**.

2.  Click **New**.

3.  On the form, fill in the fields.

</th><th>

</th></tr></thead><tbody><tr><td>

Toggle

</td><td>
