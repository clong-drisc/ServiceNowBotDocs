# Create a dynamic attribute group

Organize related attributes into a dynamic attribute group.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Dynamic Schema** &gt; **Dynamic Attribute Groups**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Label|Label used for displaying the dynamic attribute group.|
    |Name|Internal name for the dynamic attribute group.|
    |Description|A brief summary of what the dynamic attribute group contains.|
    |Active|Option to activate the dynamic attribute group.|

4.  Select **Submit**.

Add more dynamic attribute groups as needed. For example, you might add dynamic attribute groups for describing television features like Screen Resolutions or Sound Formats.
