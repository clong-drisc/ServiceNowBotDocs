# Request translations in Localization Workspace: Scope

Select specific documents to add to a translation request in Localization Workspace.

## Before you begin

Role required: localization\_requestor

## About this task

You can select or deselect specific documents for translation from the list of documents that Localization Workspace retrieved in the previous step.

The following procedure covers step three of four steps in the Translation Request wizard.

## Procedure

1.  Complete steps one and two in the Translation request wizard, then proceed to this one, **Scope**.

    Localization Workspace retrieves all untranslated or partially translated documents according to the content types that you selected in the previous step. By default, all retrieved documents are selected for inclusion in your translation request.

2.  Reveal the Types selector by opening the **Resizable panes divider**, then expand Types by selecting the carat icon. ![The Types step in the Translation Request wizard in Localization Workspace, with the Knowledge type highlighted.](../image/lw-request-translations-scope1.png)

3.  Deselect any retrieved documents that you don't want to include in your request.

    It's possible to select a document for one target language, and deselect the same document for a different target language.

4.  Select **Save**.

5.  Select **Next** to proceed to the next step in the wizard, **Estimate**.


## What to do next

Advance to the next step in the Translation Request wizard, **Estimate**.
