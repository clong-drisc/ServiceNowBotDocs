# Assign a role to a group

You can assign a role to a group to grant access to applications and modules to group members.

## Before you begin

Role required: user\_admin or admin

## About this task

When you assign roles to groups rather than to individual users, members of the group inherit the role. When a user switches groups, the new group role is assigned automatically. For information about the Service Mapping roles, see [Control user access to application services](https://www.servicenow.com/docs/access?context=control-user-access-to-business-services&version=yokohama&pubname=yokohama-it-operations-management&ft:locale=en-US).

## Procedure

1.  Navigate to **All** &gt; **User Administration** &gt; **Groups**.

2.  Select the group to assign a role.

3.  In the **Roles** related list, select **Edit**.

4.  Add the desired roles to the group.

5.  Select **Save**.
