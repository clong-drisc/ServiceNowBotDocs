# Proxy servers for SOAP clients

Administrators can specify separate proxy settings for SOAP clients, such as the MID Server or ODBC Driver.

To specify a proxy server for a MID Server, see [MID Server configuration](https://www.servicenow.com/docs/access?context=c_MIDServerConfiguration&version=yokohama&pubname=yokohama-servicenow-platform&ft:locale=en-US)

To specify a proxy server for the ODBC driver, see [Configure ODBC to use proxy servers](https://www.servicenow.com/docs/access?context=configure-odbc-proxy-servers&version=yokohama&pubname=yokohama-api-reference&ft:locale=en-US)
