# Localization Framework reference

Reference topics provide information on components and technical aspects of the Localization Framework.

-   **[Translation modes](translation-modes.md)**  
The Localization Framework offers various translation modes for content translation. Localization fulfillers can use one of the translation modes to translate content and fulfill localization requests.
-   **[States of localization projects and tasks](../reference/localization-task-states.md)**  
The state of a localization project or of a task defines the status of the localization request. The states of the localization tasks change based on the workflow that is defined for translations.
-   **[Error messages in the Localization Framework](../reference/error-messages.md)**  
Error messages are displayed when the appropriate settings are not configured for the translation and workflow preferences.
