# Activate or deactivate an EVAM view configuration

Activate or deactivate the Entity View Action Mapper \(EVAM\) view configurations in the AI Search application as needed. By activating or deactivating an EVAM view configuration, you can control which view configurations are in use.

## Before you begin

Role required: admin or ais\_admin

## About this task

When you configure a new EVAM view configuration for your AI Search application, you must manually activate it to enable it.

## Procedure

1.  Navigate to **All** &gt; **AI Search Admin** &gt; **AI Search Admin Home**.

2.  On the **Applications** tab, select the application card that you want to activate or deactivate.

3.  On the sidebar tab, select **Result-card Interface**.

4.  From the Select a bundle to edit list, select an EVAM configuration bundle.

    The list of view configurations that are associated with the selected bundle are displayed.

5.  Hover over the view configuration that you want to activate or deactivate, select the More Actions icon \(![](../../localization-framework/image/more-actions-icon.png)\), and then select **Activate or deactivate**.


## Result

The selected view configuration is activated or deactivated.
