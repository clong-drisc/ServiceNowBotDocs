# View the Dependency Views map for CMDB alerts in Express List

The Dependency Views map provides insights into how and why alerts are grouped, supporting both troubleshooting and proactive management. It displays the connecting CIs among CMDB alerts within a group. Even if a CI isn’t part of the group, it appears on the map if it connects to other alert CIs within the group.

## Before you begin

Role required: evt\_mgmt\_user, evt\_mgmt\_operator

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  In the primary navigation, select either the List icon \(![List icon](../image/list-icon-sow-itom.png)\) or the Express List icon \(![Express List icon](../../event-management/image/express-list1.png).

3.  For an alert for which you want to see the CI dependency view, in the Configuration items columns, select the configuration item.

4.  In the configuration items window, select **Open Dependency View**.

    You can see the dependency view of the CMDB alerts.
