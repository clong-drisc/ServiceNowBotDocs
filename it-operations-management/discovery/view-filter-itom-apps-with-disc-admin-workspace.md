# View and filter ITOM Visibility apps using the Discovery Admin Workspace

Access all ITOM Visibility apps from a single location in the Discovery Admin Workspace. View and filter both the applications already installed on your instance and those applications available for installation. Use this feature to integrate applications and enhance the Discovery experience.

## Before you begin

Role required: discovery\_admin or admin

This page enables you to install new applications and update existing ones. Updating applications requires you to have the admin role.

## About this task

While you can access details about the apps installed on your instance, information regarding pricing and packages isn’t provided, as it varies based on each contract. For a general overview of licensing and subscription details, see [ITOM/OT SU Licensing and subscriptions](../../it-operations-management/reference/itom-su-licensing-landing-page.md).

## Procedure

1.  Navigate to **Workspaces** &gt; **Discovery Admin Workspace**.

2.  Select **View all** in the ITOM Visibility apps section.

    The **Visibility applications** page opens displaying all applications separated into two categories: **Installed applications** and **Not installed applications**.

3.  Use the filter drop-down list to select the desired subscription for viewing.
