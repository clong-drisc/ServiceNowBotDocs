# Advanced data input configuration

When you have configured a data input successfully, Health Log Analytics adds a record to the Data Inputs table and attaches the configuration file to it. You can configure advanced settings for your data input. Configuring advanced settings is optional.

-   **[Configure advanced settings for Rsyslog, Splunk, or TCP data inputs](../task/hla-data-input-adv-rsyslog-splunk.md)**  
Configure advanced settings for data inputs that use Rsyslog, Splunk, or TCP agents.
-   **[Configure advanced settings for Beats data inputs](../task/hla-data-input-adv-beats.md)**  
Configure advanced settings for data inputs that use Beats agents.
-   **[Configure advanced settings for Elasticsearch data inputs](../task/hla-data-input-adv-elastic.md)**  
Configure advanced settings for data inputs used for streaming log data from Elasticsearch indices to your instance.
-   **[Configure advanced settings for MID Server data inputs](../task/hla-data-input-adv-mid-server.md)**  
Configure advanced settings for data inputs that are used for collecting and streaming MID Server log messages.
-   **[Configure advanced settings for Amazon CloudWatch data inputs](../task/hla-data-input-adv-cloudwatch.md)**  
Configure advanced settings for data inputs used for streaming log data from Amazon CloudWatch to your instance.
-   **[Configure advanced settings for Amazon S3 data inputs](../task/hla-data-input-adv-s3.md)**  
Configure advanced settings for data inputs used for streaming log data from Amazon S3 buckets to your instance.
-   **[Configure advanced settings for Microsoft Azure Log Analytics data inputs](../task/hla-data-input-adv-azure-la.md)**  
Configure advanced settings for data inputs used for streaming log data from Microsoft Azure Log Analytics to your instance.
-   **[Configure advanced settings for Microsoft Azure Event Hubs data inputs](../task/hla-data-input-adv-event-hubs.md)**  
Configure advanced settings for data inputs used for streaming log data from Microsoft Azure Event Hubs to your instance.
-   **[Configure advanced settings for Apache Kafka data inputs](../task/hla-data-input-adv-kafka.md)**  
Configure advanced settings for data inputs used for streaming log data from Apache Kafka to your instance.
