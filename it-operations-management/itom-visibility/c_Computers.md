# Operating systems discovery

Discovery identifies the following computers, clusters, and virtual machines.

-   **[AIX server discovery](../reference/r_DataCollDiscoAIXComputers.md)**  
Discovery identifies and classifies information about AIX servers.
-   **[Apache HBase instance discovery](../reference/r_DiscoverHBaseInstances.md)**  
Discovery creates or updates a CMDB record when it detects a running instance of HBase on a UNIX server.
-   **[HP-UX discovery](../reference/r_DataCollDiscoHPUXComputers.md)**  
Discovery identifies and classifies information about HP-UX computers.
-   **[Hyper-V discovery](../reference/r_DiscoveryForHyperV.md#)**  
The ServiceNow Discovery application finds Microsoft Hyper-V hypervisors in your environment.
-   **[Red Hat JBoss server discovery](c_DataCollDiscoJBossServers.md)**  
Discovery can detect JBoss application servers running on Linux and Windows systems.
-   **[Linux discovery](../reference/r_DataCollDiscoLinuxComputers.md)**  
Discovery and Service Mapping applications use probes and patterns to discover and map information about Linux computers and servers. The information is populated in the CMDB. Discovering some of these resources may require updating to the latest version of the Discovery and Service Mapping Patterns application from the ServiceNow Store.
-   **[Netware discovery](../reference/r_DataCollDiscoNetware.md)**  
Discovery identifies and classifies information about Netware.
-   **[Mac \(OS/X\) discovery](../reference/r_DataCollDiscoMacOSXComputers.md)**  
Discovery identifies and classifies information about Mac \(OS/X\) computers.
-   **[Solaris discovery](../reference/r_DataCollDiscoSolarisComputers.md)**  
Discovery identifies and classifies information about Solaris computers.
-   **[Discovery for data-center virtualization](../reference/r_DataCollDiscoVirtualMachines.md)**  
Discovery and Service Mapping Patterns identify and classify information about virtual machines discovered in a data-center.
-   **[Discovery for VMware virtualization](c_DiscoverVMwareInfrastructure.md)**  
A Discovery schedule for VMware virtualization can discover vCenter and ESX hosts as well as individual ESXi hosts that manage VMs and related components without a vCenter.
-   **[Windows discovery](../reference/r_DataCollDiscoWindowsComputers.md)**  
Discovery identifies and classifies information about Windows computers that use IPv4 addresses, IPv6 addresses, or both.
-   **[z/OS discovery](zOS-discovery.md)**  
Discovery and Service Mapping Patterns finds computers running the z/OS operating system using the IBM zOS Server pattern. Discovering some of these resources may require updating to the latest version of the Discovery and Service Mapping Patterns application from the ServiceNow Store.
