# Viewing the requester dashboard

View and manage cloud account requests on requester dashboard. The dashboard provides a view of requested, approved, denied, pending, and active accounts.

You can access the dashboard in either of two ways.

-   Navigate to **All** &gt; **Cloud Workspace** &gt; **Home**
-   Navigate to **Workspaces** &gt; **Cloud Workspace** &gt; **Home**
