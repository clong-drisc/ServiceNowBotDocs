# Agent Client Collector Framework

The Agent Client Collector Framework \(ACC-F\) application manages the agent installation on your infrastructure components, enabling sending of output data to the ServiceNow® instance via the MID Server. In addition to the MID Server, you must install the Configuration Management Database for Scoped Apps \(CMDB\).
