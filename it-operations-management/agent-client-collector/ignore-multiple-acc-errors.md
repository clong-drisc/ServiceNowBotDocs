# Ignore multiple Agent Client Collector errors

Mark multiple Agent Client Collector errors as **Ignored** when the errors are expected or reflect an issue that is not important for you to resolve.

## Before you begin

Role required: agent\_client\_collector\_user, itil

## Procedure

1.  Navigate to **All** &gt; **Agent Client Collector** &gt; **Agent Issues**.

    The **ACC Error Messages** page appears.

2.  Hover underneath the selection box ![Selection box](../image/selection-box.png) next to the relevant error.

3.  Select the box that appears next to each error you want to ignore.

4.  Select the drop down arrow in the **Actions on selected rows...** cell.

5.  Select **Ignore Error**.

    All of the selected errors take on the state **Ignored**.

    To remove the Ignored state from errors, select the check box next to the relevant errors and select **Unignore Error** from the **Actions on selected rows...** cell.
