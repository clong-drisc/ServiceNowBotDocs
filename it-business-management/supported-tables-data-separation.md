# Supported tables for Data Separation

Several types of tables are supported for Data Separation, including direct and indirect related tables for each entity.

You can enable data separation on the following entities and related entities.

-   Entities:
    -   Demand \[dmn\_demand\]
    -   Project \[pm\_project\]
    -   Resource Plan \[resource\_plan\]
    -   Cost Plan \[cost\_plan\]
-   Related entities:
    -   Cost Plan Breakdown \[cost\_plan\_breakdown\]
    -   Project Task \[pm\_project\_task\]
