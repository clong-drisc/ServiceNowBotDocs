# Event registry

The events registry lists the events the system recognizes. Use registered events to automate other activities, such as script actions or notifications.

After you create a new event and a business rule that uses the event, you must register it.

Registration lets other parts of the system, such as [Email and SMS notifications](https://www.servicenow.com/docs/access?context=c_EmailNotifications&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US) and Script Actions, see the event in their list of available events and react to the event when it occurs.
