# Summarize a flow

Generate a summary of a flow with Now Assist for Creator.

## Before you begin

Role required:

-   now.assist.creator
-   flow\_designer or admin

## Procedure

1.  Navigate to **All** &gt; **Process Automation** &gt; **Workflow Studio**.

2.  On the homepage, select **Flows**.

3.  Open the flow that you want to summarize.

4.  From the More Actions menu, select **Flow summary**.

5.  From the Flow summary dialog box, select **Summarize**.

    ![An example summary of the Change - Standard flow.](../images/flow-summary-03.png)


## Result

Now Assist for Creator generates a summary of the flow. The flow summary also displays its creation date.

## What to do next

You can regenerate the flow summary by selecting the **Refresh summary** icon next to the summary creation date.
