# Subflow activities in workflow

Subflow activities run and manage workflows from a parent workflow.

The [Parallel Flow Launcher](https://www.servicenow.com/docs/access?context=r_ParallelFlowLauncher&version=yokohama&pubname=yokohama-servicenow-platform&ft:locale=en-US)subflow activity is available.

-   **[Parallel Flow Launcher workflow activity](../reference/r_ParallelFlowLauncher.md)**  
The **Parallel Flow Launcher** activity launches multiple subflows in parallel.
