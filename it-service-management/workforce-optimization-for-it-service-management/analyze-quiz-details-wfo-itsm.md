# Analyze quiz details in Workforce Optimization for ITSM

Access a quiz to analyze the details.

## Before you begin

Role required: sn\_wfo\_cfg\_itsm.manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Manager Workspace**.

2.  Select the coaching icon.

3.  Select the **Learning** tab.

4.  Go to **Learning courses** &gt; **Internal courses**.

5.  Select a quiz.

6.  Select **View quiz details**.

    The quiz details page appears.
