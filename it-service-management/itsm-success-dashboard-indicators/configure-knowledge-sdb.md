# Configure the knowledge base for ITSM Success Dashboard indicators

Include only the relevant knowledge bases related to IT while determining the **Self-solved using Knowledge** KPI in ITSM Success Dashboard indicators.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Self-Service Analytics** &gt; **Activity Context**.

2.  Select **User activity** from the list.

3.  In the **Activity Context Types** tab, select the Preview icon corresponding to Viewed a knowledge article activity type.

4.  Select **Open record**.

5.  In the Activity Context Type form, change the filter condition to include the relevant knowledge bases.

6.  ![activity context type](../image/activity-context-type.png "Activity context type")

7.  Select **Update**.
