# Components installed with DEX Self-service

Several tables are installed along with DEX Self-service.

## Tables installed

|Table|Description|
|-----|-----------|
|DEX self-service categories|Stores the categories and sub-categories which are marked as Good, Average, or Poor in the Device health check experience.|
|DEX self-service issue configs|Stores the issues which are associated with categories, evaluation criteria, metrics, and resolutions, based on the monitored device with the resolutions appearing for each issue.|
|DEX self-service device actions|Stores actions which are not related to a specific issue.|
