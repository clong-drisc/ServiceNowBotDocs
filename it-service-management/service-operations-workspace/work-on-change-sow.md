# Work on a change request in Service Operations Workspace

Modify a supported configuration item \(CI\) if required and close the change request.

## Before you begin

Role required: itil

## Procedure

1.  Open a change request.

    For information about creating a change request in Service Operations Workspace, see [Create a change request in Service Operations Workspace](create-change-sow.md).

2.  Perform any of the following actions on the change record page.
