# Edit delivery channel

Edit rules and delivery channels.

## Before you begin

Role required: rota\_manager, rota\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operation Workspace**.

2.  Select the **Schedule** icon.

3.  On **Schedules** page, select **Notification preferences** and select **Edit** to add or edit Delivery channels.

4.  Select **Add New** to add a new channel and provide name and email ID.

5.  Select **Phone numbers** tab to add and edit phone numbers.

6.  Select **Save** to apply your changes.
