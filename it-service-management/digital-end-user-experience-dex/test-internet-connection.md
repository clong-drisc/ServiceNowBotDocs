# Test the internet connection of your system

Conduct an internet speed test to evaluate the performance of your internet service. If it seems like your internet is slower than it should be, running a speed test is the quickest way to find out if something is wrong on your end, or if the problem is with your internet service provider.

## Before you begin

Confirm that the DEX Desktop Assistant \[sn\_dex\_desktop\] application has been installed. For more information, see [Install Application and Device Health](install-app-device-health.md) and [Download and install Desktop Assistant](download-desktop-experience.md).

Role required: sn\_dex\_desktop.user or sn\_dex\_desktop.admin

## Procedure

1.  On the Desktop Assistant home page, click the **Network Test** card.

    The Network test page opens.

    ![The Network test page where you can test the network speed of the device.](../image/desktop-exp-network-test-1.png)

2.  Click **Test my network**.

    The test provides the network speed \(download and upload speed\) of the device where the Desktop Assistant is installed. Under the Network Details section, you can find the machine name and the server details.

    ![The Network test page that gives the details of your network speed.](../image/desktop-exp-network-test-2.png)

    If the network speed is below average, you can check the details for the troubleshooting steps.
