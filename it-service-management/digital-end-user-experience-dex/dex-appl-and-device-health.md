# DEX application and device health

Use DEX Application and Device Health to monitor and look up applications and device metrics.

-   **[Monitor system performance](../task/monitor-system-performance.md)**  
Monitor your organization's system performance effectively by tracking CPU, memory, disk, and I/O metrics. This helps prevent application bottlenecks and verifies resource availability. Disk performance monitoring identifies potential storage issues, such as slow read or write speeds, while I/O monitoring helps detect delays in data transfer.
-   **[View actions in the Digital End-User Experience Action Library](../task/view-actions-in-actions-library.md)**  
View and use various pre-installed services available for a device in the Digital End-User Experience Action Library.
-   **[Monitor your user apps network](../task/monitor-user-apps-network.md)**  
Monitor your app's network to ensure top performance, reduce latency, and improve security. Track key metrics to spot issues early, optimize resource use, and fix problems before they impact users.
-   **[View metrics of an installed application](../task/view-installed-app-metrics.md)**  
View and monitor the CPU and memory usage of your applications to optimize performance and promoting efficient resource allocation.
-   **[View metrics of a web application](../task/view-web-app-metrics.md)**  
View metrics of a web application to get a comprehensive view of the performance of the application. Tracking these metrics over time can help you gauge the impact of changes made to the application and monitor its long-term performance.
-   **[View collected metrics with Metrics analyzer](../task/view-dex-metrics.md)**  
Use the Metrics analyzer to view metrics collected for a given device or application during a specific time.
-   **[View client health with MCM](../task/view-client-health-with-mcm.md)**  
View client health metrics with the Microsoft Configuration Manager \(MCM\) application. The metrics include MCM metrics by device count, MCM versions installed on user devices, and device count where MCM action hasn't run in the last 24 hours.
-   **[View the domain or URL for a web application](../task/view-app-url.md)**  
View the domain or web address of a web application to access the application and use it effectively.
-   **[View the support group for an application](../task/view-app-support-group.md)**  
View the support group for the application to identify who to contact when you encounter issues. The support group is typically responsible for managing incidents, troubleshooting problems, and providing technical support for the application.
-   **[View agent policies](../task/view-agent-policies.md)**  
View the agent policies used during check runs on the associated Configuration Items \(CI\).
-   **[Monitor Microsoft Teams call quality](../../dex-for-microsoft-365/task/monitor-teams-call-quality.md)**  
Enable advanced monitoring of Microsoft Teams call quality with DEX for Microsoft 365. Track and analyze call performance for a specific user.
-   **[Monitor Zoom call quality using DEX](../Zoom-calls-monitoring/monitor-zoom-call-quality-using-dex.md)**  
View detailed metrics for the Zoom calls made by users assigned to this device or logged in within the past 15 days.
-   **[View Zoom room metrics](../Zoom-calls-monitoring/view-zoom-room-metrics.md)**  
View detailed metrics for your organization's Zoom rooms using DEX.
