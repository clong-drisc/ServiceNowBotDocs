# View system compliance

View whether the devices are adhering to your organization's application and policy compliance rules. Use this data to identify potential security risks.

## Before you begin

Confirm that the DEX plugin \(sn\_dex\) is installed.

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  In the primary navigation pane, select the Insights icon \(![Insights icon](../image/icon-insights.png)\).

3.  Select **System compliance**.

    For more information on the system compliance metrics, see [DEX Insights reports](../reference/insights-reports.md).
