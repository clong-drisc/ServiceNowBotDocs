# Accessing Desktop Assistant

Get insights across different Desktop Assistant features by exploring components within the application.

![Desktop Assistant home page](../image/DA_Home.png "Desktop Assistant Home page")

<table id="table_r2b_mxq_lwb"><thead><tr><th>

Components

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Header

</td><td>

Header includes the name of the home page, the notification icon and the user icon. The notification bell icon \(![Notification bell icon](../image/icon-notification-bell.png)\) displays a small red dot with number to indicate the number of new notifications. The user's profile icon \(![User's profile icon that can be used to login and logout from Desktop Assistant.](../image/icon-user-login.png)\) is the user log off icon.

</td></tr><tr><td>

Sections

</td><td>

Sections make it easy for you to search for items by arranging items sequentially and in logical groups. By default, ServiceNow provides My Resources.Desktop Assistant administrator can configure the section. For more information, see [Add a section in the Desktop Assistant home page](../task/add-home-page-section.md).

</td></tr><tr><td>

Cards

</td><td>

Application shortcuts that help you to access applications quickly.Desktop Assistant administrator can map a card to any section or remove cards from any section. For more information, see [Delete a card from the Home page](../task/delete-card.md).

</td></tr></tbody>
</table>
