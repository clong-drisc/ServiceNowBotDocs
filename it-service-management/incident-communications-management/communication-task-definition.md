# Communication task definition

Each communication plan involves a series of communication tasks. A communication task definition helps you to specify the mode of communication \(channel\) and the frequency at which the communication must be carried out.

For information on how to create a communication task definition, refer to [Create communication task definition](https://www.servicenow.com/docs/access?context=create-comm-task-definition&version=yokohama&pubname=yokohama-servicenow-platform&ft:locale=en-US).
