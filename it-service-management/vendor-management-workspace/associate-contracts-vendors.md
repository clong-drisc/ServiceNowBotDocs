# Associate contracts with vendors

Create a contract and associate it with the vendor who is responsible for the contract.

## Before you begin

Role required: sn\_vlm.vendor\_manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Vendor Management Workspace**.

2.  Select the List icon.

3.  Go to **Contracts** &gt; **All Vendor Contracts**.

4.  Select **New**.

5.  Fill in the fields on the form.

    For information on the field descriptions, see [Create a contract](https://www.servicenow.com/docs/access?context=t_CreateAContract&version=yokohama&pubname=yokohama-it-asset-management&ft:locale=en-US).

    **Note:** The start date must be the current day or a day before the current day.

6.  Select **Save**.

    In the List menu, go to **Contracts** &gt; **My Vendor Contracts** to view the contract you've created.
