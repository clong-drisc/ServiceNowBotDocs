# Success case playbook

Use the success case playbook as the starting point to define the success processes in your organization. This playbook can contain one or more success case tasks with predefined activities.

The success case playbook consists of three stages:

-   Initiate: Create a success case record for an engagement and enter the core information.
-   Assist: Create success tasks related to the success case. In this stage, you can define the intended outcome, action plan, and create meetings.
-   Close: Close the playbook when all the success tasks have been completed.
