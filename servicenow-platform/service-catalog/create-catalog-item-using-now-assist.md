# Creating a catalog item using Now Assist

Create a catalog item by using Now Assist and make the item available in Service Catalog for catalog item requests. You can also define dynamic behavior for your catalog item and apply auto-populate settings.

## Before you begin

Role required: catalog\_builder\_editor and now.assist.creator

## Procedure

1.  Navigate to **All** &gt; **Service Catalog** &gt; **Catalog Builder**.

2.  Select the **Dashboard** tab and select **Create a new catalog item**.

3.  Review the steps involved in the creation process and select **Continue**.

4.  Select a template from the list and select **Use this item template**.

5.  In the Now Assist step, describe the catalog item you want.

    For more information about describing your catalog item, see [How to describe a catalog item](../reference/how-to-describe-catalog-item.md).

6.  Select **Try an example** to view sample prompts for creating catalog items using Now Assist.

    **Note:** Select **Try an example** again to view more samples. You can edit the text in the sample and describe the item as per your requirements.

7.  Select **Generate with Now Assist**.

    Now Assist generates content for the Details and Questions steps of the catalog item.

8.  Verify the content in the Details and Questions steps before you submit the item.

    **Note:** The AI sparkle \(![AI sparkle icon.](../../../common/image/icon-ai-sparkle.png)\) icon displayed for these steps and their fields denotes that the content in the fields is AI-generated. If you change the content, the sparkle is removed.
