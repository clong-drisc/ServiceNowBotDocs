# Wrap up codes

Define the wrap up codes.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Interaction** &gt; **Wrap Up Codes**.

    The Interaction Wrap Up Codes screen appears.

2.  Do one of the following:

    -   To edit an existing code, select the name of the code.
    -   To create a new code, select **New**.
3.  Fill in the fields:

    |Field|Description|
    |-----|-----------|
    |Code|Name of the wrap up code.|
    |Application| |
    |Active|Indicates whether wrap up code is enabled.|
    |Description|Description of the wrap up code.|

4.  Select **Update** if you are updating an existing code or **Submit** if you are creating a new code.
