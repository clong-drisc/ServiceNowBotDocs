# Monitoring and managing service instances

The Service Instances dashboard \(formerly Application Services\) enables you to monitor and manage service instances to ensure that they are fully configured and are populated in the CMDB.

## Working in the dashboard

A report on the dashboard, for example, can direct you to a service instance that is not configured with a service population method so you can repair it. For information on working in the dashboard, see [Monitor the health of application services on the Application Services dashboard](../../configuration-management/task/app-service-dashboard.md).
