# Specifying attributes and relationships for service instances

Use the Service Instance Settings module to specify the attributes and relationships that are required when a user creates a service instance.

## Configuring service instance settings

Navigate to **CSDM** &gt; **Service Instance Settings** to access the form. For instructions, see [Modify the attributes and relationships required for application services](../../configuration-management/task/modify-req-fields-appservice.md).
