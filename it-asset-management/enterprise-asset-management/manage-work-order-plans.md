# Manage asset-centric work tasks using work order plans

Increase operational efficiency by organizing complex asset-centric work tasks such as shutdown, maintenance, and calibration, by using work order plans.

**Important:** Work order plan templates and work order plans are available with Enterprise Asset Management version 10.0 and later.

Create reusable work order plan templates and use them when creating work order plans. For more details, see [Asset work order plans in Enterprise Asset Management](work-order-plans-eam-assets.md).

-   **[Create a work order plan template](../task/creat-work-plan-template.md)**  
Create a reusable work order plan template to use when creating work order plans.
-   **[Create a work order plan for enterprise assets](../task/create-work-order-plan-eam.md)**  
Create and manage a work order plan to organize, assign, schedule, and track complex asset tasks for enterprise assets and asset groups.
